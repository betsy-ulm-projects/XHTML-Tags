CSS Lectures

Cascading Style Sheets (CSS) provides support for all HTML formatting.
It can be defined three ways:

  1. Inline style: per HTML element
  2. Embedded style: within an HTML file
  3. External style: within an external file
Div elements are preferred over p tags.

Inline style attributes (inside div tag):
 style="property: value"
NOTE: property:value is always a pair.
These are placed between body tags.

Embedded style attributes (inside style tag):
 opening style tag type="text/css" end
 elementname1 {property: value; property1: value1}
 closing style tag
This type goes between the head tag; it allows a global format.

External style attributes (inside another file):
 elementname1 {property: value; property1: value1}
 elementname2 {property: value; property3: value3}
Calling the CSS from inside the XHTML file:
  opening head tag
  begin link rel="stylesheet" type="text/css" href="fileLocation.css" end
  closing head tag

       Inline Style
        begin element name style=“background-color: value” end
        Embedded and External Style
        element name {background-color: value}

  6. Positioning:
    • Static positioning is the default
    • Relative positioning is when the content is offset relative to its normal position. It's specified with these attributes
      1. top
      2. botton
      3. right
      4. left
    • Absolute positioning: offset; this one is the worst because it overlaps elements and it's cancerous looking


  7. Numeric positioning:
    • top: 125px
    • left: 0px
    • also right and bottom
    • z-index: #
       This specifies the stack order of an element. Larger whole numbers are always in front of smaller whole numbers.

  8. Span elements: these allow you to edit inside a line or block of text without adding unwanted space or new lines.
