Tables:

Basic table structure:

Using a border of 1px. Default is no border.
Cell spacing between other cells 20px. Default is 0px.
Cell padding is the amount of space between the border of the cell and it's contents 2.5px. Default is 0px.
Width of frame around the table 250px. Default is determined by the user agent.

The th tag center aligns and bolds text. The td tag only left aligns the text.

Forms:

Basic form structure:


Form tag attributes:

  1. action = "specifies a form processing agent i.e. URI"
  2. method = "specifies which HTTP method will be used to submit the form data set i.e. get, post"
    • "get" method: the form data is appended to the URI specified by the action attribute and this new URI is sent to the processing agent
    • "post" method: the form data is included in the body of the form and sent to the processing agent

The input tag specifies the type of input (i.e. hidden, text, radio, image)
The attributes for input tags are their type="whatever" and value="what you want to show on the button"

Input types and their attributes:
  • EMPTY ELEMENT!   input type=["whatever","whatever2"]
  • reset: resets all controls to their inital values
  • submit: submits a form
  • hidden: not rendered on the form but still submitted with a form
  • text: allows users to input text in a single line input control
    1. value="the placeholder value"
    2. size="width of bar"
    3. maxlength="max characters the user can input"
  • password: similiar to text, but the characters typed are hidden
  • checkbox: allows you to select one or more items from a list
  • radio: only allowed to select one item from the list; mutually exculsive
